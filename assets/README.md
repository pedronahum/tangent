# Tangent Assets

This directory contains visualization examples and code outputs used in the main README.md.

## Contents

### Visualization Images (PNG)

1. **computation_graph.png** - Shows how functions are decomposed into computation graphs
2. **gradient_flow.png** - Displays forward and backward gradient propagation
3. **multivariate_flow.png** - Visualizes gradient flow for multivariate functions
4. **gradient_comparison.png** - Compares automatic differentiation vs numerical gradients
5. **vector_gradients.png** - Shows gradient computation for vector-valued functions

### Code Output Examples (TXT)

1. **gradient_code_output.txt** - Example output from `tangent.show_gradient_code()`
2. **basic_usage_output.txt** - Basic usage example with terminal output
3. **numpy_example.txt** - NumPy backend example with output
4. **jax_example.txt** - JAX backend example with output
5. **tensorflow_example.txt** - TensorFlow backend example with output
6. **caching_example.txt** - Performance comparison showing caching benefits
7. **multiple_gradients_example.txt** - Multivariate gradient example with output

## Regenerating Assets

All assets can be regenerated by running:

```bash
python3 generate_readme_assets.py
```

This script will:
- Generate all PNG visualizations using Tangent's visualization tools
- Create text files with actual code execution outputs
- Overwrite existing files

## Usage in README

These assets are referenced in the main README.md to:
- Show real visualization outputs instead of just describing them
- Provide actual execution results for code examples
- Make the documentation more engaging and professional

## File Sizes

Total size: ~310 KB
- PNG images: ~307 KB
- Text files: ~3 KB

All images are optimized for web viewing (150 DPI) while maintaining good quality.
